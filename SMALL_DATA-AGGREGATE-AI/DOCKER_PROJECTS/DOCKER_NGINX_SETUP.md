# Docker NGINX project

## Docker setup
```console
sudo apt update
sudo apt upgrade
sudo apt install docker
```
## Docker Compose setup
```console
sudo apt install docker-compose
docker–compose –version
```
##  Docker Compose NGNIX setup
```console
cd dockerbuild
docker-compose.yml
```

```console
cd /dockerbuild/ngninxbuild
nano Dockerfile
```
